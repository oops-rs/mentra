# mentra v0.7.1

Mentra `v0.7.1` is a compatibility patch for downstream workspaces that also
use newer sqlite bindings.

## Compatibility

- Update `rusqlite` from 0.32.1 to 0.39.
- This aligns Mentra with the `libsqlite3-sys` 0.37 linkage family, avoiding
  Cargo `links = "sqlite3"` conflicts when Mentra is composed with crates that
  already depend on current `rusqlite`.

## Repository Hygiene

- Ignore local `.grapha` graph artifacts.

## Crate Versions

- `mentra 0.7.1`
- `mentra-provider 0.3.0` remains unchanged and does not need to be republished.

## Verification

This release is expected to pass:

- `cargo fmt --all --check`
- `cargo clippy --workspace --all-targets -- -D warnings`
- `cargo test --workspace`
